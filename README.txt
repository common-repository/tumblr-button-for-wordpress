This plugin makes it very easy to add the Tumblr Button to the botton of each of your individual post pages.

== Description ==

Tumblr Button For WordPress allows you to easily incorporate the Tumblr Share Button into the bottom of each of your post pages. 

After installation, simply select your favorite button, save, and done!

<a href="http://tommcfarlin.com/tumblr-button-for-wordpress">Read more...</a>

== Installation ==

1. Upload the plugin to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Locate the 'Tumblr Button' icon in the WordPress menu
4. Select your favorite button, save, and blog!
